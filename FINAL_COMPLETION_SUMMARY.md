## 🎮 使用指南

### 🚀 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/y6833/elp.git
cd elp

# 2. 安装依赖
npm install

# 3. 设置关卡环境
npm run setup:levels

# 4. 启动学习平台
npm start

# 5. 开始学习
# 访问 http://localhost:3000
```
